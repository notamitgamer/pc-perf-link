# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it immediately to ensure the safety of our users.

### Immediate Contact
For critical security concerns, email: amitdutta4255@gmail.com

### Response Time
We aim to acknowledge critical security reports within 24-48 hours and provide a plan for resolution as quickly as possible.

### General Inquiries
For non-critical bugs or general security suggestions, you may use the standard support email: mail@amit.is-a.dev

# Contributing to pc-perf-link

We welcome contributions to improve the performance and features of this project.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Ensure your code follows the existing style and is well-commented.
4. Submit a pull request with a detailed description of your changes.

## Bug Reports and Suggestions

If you find a bug or have a suggestion, please send an email to mail@amit.is-a.dev. 
For security-related issues, please refer to the SECURITY.md file.
